  `345455535455535455335555555544`
- `3=F1`, `4=F2`, `5=AŞAĞI`

- GP9 makroyu bekleme süresini atlayarak hemen çalıştırır.

- F8, GP19, kalibrasyon, ortak kredi, 0–10 titreme filtresi ve 2 dakika röle
  uykusu korunmuştur.
