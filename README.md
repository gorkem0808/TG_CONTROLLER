# TG_CONTROLLER V003

V003, profesyonel koyu temalı Windows yönetim programını ve Controller firmware'ini birlikte üretir.

## Windows Manager

- Dashboard
- Controller canlı tuş testi
- Röle 1 / Röle 2 testi
- Firmware dosyası seçme
- Settings
- Canlı log ekranı
- COM port bağlantısı
- Firmware sürüm bilgisi
