# V002 Kullanım

1. GitHub Actions çıktısından `TG_CONTROLLER_V002.uf2` dosyasını indir.
2. Pico'ya BOOTSEL ile yükle.
3. Windows çıktısından `TG_CONTROLLER_MANAGER_V002.exe` dosyasını indir.
4. Programı aç, doğru COM portunu seç ve **Bağlan** düğmesine bas.
5. `PONG TG_CONTROLLER V002` görülürse bağlantı hazırdır.
6. Röle test düğmeleri röleyi yalnız 250 ms çalıştırır.
