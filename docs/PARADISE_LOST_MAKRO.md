# Paradise Lost Sabit Rakam Makrosu

Makro değiştirilemez:

```text
345455535455535455335555555544
```

Eşleştirme yalnız rakam tuşlarıdır:

```text
3 = Klavye 3
4 = Klavye 4
5 = Klavye 5
```

F1, F2 ve Aşağı tuşları bu makroda kullanılmaz.

GP9 kısa basış makroyu bekleme süresi olmadan hemen çalıştırır.
