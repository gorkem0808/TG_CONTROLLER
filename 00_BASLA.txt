5. PyInstaller paketine pystray Windows arka ucu açıkça eklenmiştir.

1. ZIP içeriğini GitHub deposuna yükleyin.
2. Actions → Build TG_CONTROLLER_PRO V4.2 TRAY HOTFIX çalışmasını açın.
3. TG_CONTROLLER_PRO_MANAGER_V4_2_WINDOWS paketini indirin.
4. TG_CONTROLLER_PRO_MANAGER_V4_2.exe dosyasını çalıştırın.
