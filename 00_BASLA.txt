TG_CONTROLLER_PRO V4.1 HOTFIX — KURULUM

BU SÜRÜMDE DÜZELTİLEN HATA
- Açılışta görülen "ModuleNotFoundError: No module named 'psutil'" kaldırıldı.
- psutil bağımlılığı projeden tamamen çıkarıldı.
- Oyun çalışıyor mu kontrolü Windows'un yerleşik işlem API'siyle yapılır.
- GitHub, oluşturduğu EXE'yi --self-test ile gerçek olarak çalıştırmadan artifact üretmez.

YÜKLEME
1. Bu ZIP'in içindeki tüm dosya ve klasörleri GitHub deposuna yükleyin.
2. .github görünmezse WORKFLOW_KOPYASI_BUILD_PRO_V4_1.yml içeriğini
   .github/workflows/build.yml dosyasına yapıştırın.
3. Actions → Build TG_CONTROLLER_PRO V4.1 HOTFIX çalışmasını açın.
4. Build Windows Manager yeşil olmalıdır.
5. TG_CONTROLLER_PRO_MANAGER_V4_1_WINDOWS paketini indirin.
6. TG_CONTROLLER_PRO_MANAGER_V4_1.exe dosyasını çalıştırın.
