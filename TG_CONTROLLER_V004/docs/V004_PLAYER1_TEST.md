# V004 Player 1 Testi

1. `TG_GUN_P1_V004.uf2` dosyasını ikinci Pico'ya yükle.
2. X pot orta ucunu GP26'ya bağla.
3. Y pot orta ucunu GP27'ye bağla.
4. Potların iki dış ucunu 3V3 ve GND'ye bağla.
5. Windows Manager V004'ü aç.
6. Player 1 Pico'nun COM portunu seçip bağlan.
7. Player 1 ekranında X ve Y değerlerinin 0-4095 arasında değiştiğini kontrol et.
8. Hareket Aç / Hareket Kapat düğmelerini test et.
